# Introduction

This project is an application that enables real estate agents to create and manage real estate ads.

# Project installation

## Required

-   XAMPP
-   Composer
-   Node.js

## Installation steps

`npm install`
`composer install`

# Project running

`composer run dev`

# Useful commands

## Sample to create a migration:

`php artisan make:migration migration-name`

## Run the migrations:

`php artisan migrate`

## Drop database:

`php artisan db:wipe`
